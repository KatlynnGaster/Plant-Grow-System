# Plant-Grow-System
Smart Plant Grow System on the ESP32 using FreeRTOS tasks and interrupts. System monitors environmental conditions and automatically controls light, water, and airflow. 

# Hardware
- ESP32 Microcontroller
- Photoresistor
- Water level sensor
- SparkFun environmental combo (BME280 & CCS811)
- External RTC module
- Servo motor
- LEDs
